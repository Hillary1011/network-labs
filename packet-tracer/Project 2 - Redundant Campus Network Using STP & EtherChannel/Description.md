Project 2: Redundant Campus Network Using STP & EtherChannel

🔧 Objective

Design a Campus network preventing loops and increase availability in the network.

🧩 Network Requirements

- 3 Switches (core + access)
- Multiple redundant links
- 1 Router for external connectivity
  
⚙️ Technical Requirements
- Static IP addressing
- Switch management IP
- Password protection (console, enable secret)
- Configure STP (PVST+)
- Set Root Bridge
- Configure EtherChannel (LACP)
- Verify port roles and states
  
✅ Validation
- STP convergence verification
- Loop prevention demonstration
- EtherChannel operational (show etherchannel summary)
  
📦 Deliverables
- .pkt file
- Network diagram
- README explaining design network
